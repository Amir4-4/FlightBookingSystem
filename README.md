#  Flight Booking System

## Project Overview
This project is a Flight Booking System developed using Java. It simulates airline booking operations such as searching flights, booking tickets, and managing reservations.

---

##  Features
- Flight search functionality
- Seat reservation system
- Ticket booking and cancellation
- Passenger management
- Simple user interface

---

##  Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Data Structures
- File Handling
